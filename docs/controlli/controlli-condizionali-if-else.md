# Controlli condizionali: If-Else in C

**Categoria:** Controlli

Fonte: `README.md`

## Descrizione

Esempi:
```c
if (x < 0) {
  /* ... */
} else if (x == 0) {
  /* ... */
} else {
  /* ... */
}
```

### Consigli
- Evitare condizioni complesse: scomporle in funzioni o variabili booleane.
