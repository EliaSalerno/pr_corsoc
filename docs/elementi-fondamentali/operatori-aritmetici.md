**Categoria:** Elementi fondamentali

Fonte: `README.md`

## Descrizione

Operatori base: `+`, `-`, `*`, `/`, `%`.
- Divisione intera tronca per interi: `5 / 2` â†’ `2`.
- `%` valido solo su interi.

### Precedenza
Multiplicativi (`*`, `/`, `%`) hanno precedenza sugli additivi (`+`, `-`).

### Esempi
```c
int a = 5 + 3 * 2; // 11
int b = (5 + 3) * 2; // 16
```
